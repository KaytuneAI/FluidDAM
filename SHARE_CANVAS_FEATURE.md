# 分享画布功能说明

## 功能概述
新增的"分享画布"按钮允许用户将当前画布状态保存为分享文件，方便与他人协作和分享作品。

## 功能特性

### 🎯 **核心功能**
- **一键分享**：点击按钮即可生成分享文件
- **完整保存**：包含所有画布数据、图片信息和页面状态
- **跨平台兼容**：生成的JSON文件可在任何设备上加载

### 🎨 **用户体验**
- **视觉反馈**：绿色按钮配发送图标（📤）
- **加载状态**：分享时显示旋转加载动画
- **成功提示**：显示绿色通知条和详细说明
- **错误处理**：失败时显示错误提示

### 📁 **文件格式**
分享文件包含以下信息：
```json
{
  "version": "1.0",
  "type": "shared_canvas",
  "sharedAt": "2024-01-01T12:00:00.000Z",
  "canvasData": { /* 完整的画布状态 */ },
  "currentPageId": "page:xxx",
  "imageInfo": [ /* 所有图片的详细信息 */ ],
  "totalImages": 5,
  "sharedBy": "Tldraw User"
}
```

## 使用方法

### 1. 分享画布
1. 完成画布创作
2. 点击右侧素材栏的"分享画布"按钮（📤）
3. 等待文件生成（显示加载动画）
4. 文件自动下载到本地
5. 将文件发送给其他人

### 2. 加载分享的画布
1. 点击"加载画布"按钮
2. 选择收到的分享文件（.json格式）
3. 画布自动恢复到分享时的状态

## 技术实现

### 组件结构
```
src/components/ShareCanvasButton.jsx
├── 状态管理（isSharing）
├── 数据收集（画布状态、图片信息）
├── 文件生成（JSON格式）
├── 下载处理（跨浏览器兼容）
└── 用户反馈（通知和提示）
```

### 依赖关系
- `tldraw`: 获取画布快照
- `downloadUtils.js`: 跨浏览器下载
- `apiUtils.js`: 获取图片信息
- `sidebarStyles.js`: 加载动画样式

## 按钮布局

现在右侧素材栏包含4个按钮：
1. **插入图片** (📷) - 添加新图片
2. **加载画布** (📁) - 加载保存的画布
3. **保存画布** (💾) - 保存当前画布
4. **分享画布** (📤) - 生成分享文件

## 样式设计

### 按钮样式
- **颜色**：绿色背景 (#28a745)
- **尺寸**：48x48像素
- **图标**：发送emoji (📤)
- **状态**：加载时显示旋转动画

### 动画效果
- **加载动画**：白色边框旋转
- **过渡效果**：按钮状态变化平滑过渡
- **禁用状态**：加载时按钮变灰且不可点击

## 错误处理

### 常见问题
1. **画布为空**：仍然可以生成分享文件
2. **图片信息缺失**：使用默认文件名
3. **下载失败**：显示错误提示
4. **API不可用**：自动回退到localStorage

### 用户提示
- **成功**：绿色通知条 + 详细说明弹窗
- **失败**：红色通知条 + 错误提示
- **加载中**：旋转动画 + 按钮禁用
