## 🔧 技术细节

### 注释方式
使用 `// console.log(...)` 而不是完全删除，便于调试时重新启用。

### 保留策略
- **错误和警告**: 全部保留
- **统计信息**: 保留重要的汇总信息
- **异常处理**: 保留异常情况的处理信息
- **调试信息**: 只在异常时输出

### 条件输出
对于某些日志，使用条件判断只在异常时输出：
```javascript
// 只在尺寸异常时输出日志
if (frameRect.width <= 10 || frameRect.height <= 10) {
  console.log(`生成图片frame: 位置(${frameRect.x}, ${frameRect.y}), 尺寸${frameRect.width}x${frameRect.height}`);
}
```
